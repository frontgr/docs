# Общие настройки форматирования

Наши общие настройки форматтера Prettier:

```json
{
  "semi": true,
  "singleQuote": false,
  "tabWidth": 2,
  "jsxSingleQuote": false,
  "brackeSpacing": true,
  "trailingComma": "all",
  "printWidth": 80,
  "endOfLine": "lf"
}
```

Создайте в корне репозитория отдельный файл .prettierrc и туда вставляете вышеуказанный JSON-код.
